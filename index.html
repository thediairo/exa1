Hemos visto cosas basicas, hypertextos que es la foma de unir dos paginas, etiquetas samanticas que pues no aportan nada relevante, solo le dan estructura, como poner contenido multimedia que es lo unico que es medianamente interesante, como subir tus paginas a la web por medio de git y git hub, empezamos a ver tambien css, que pues nada mas es para que este bonita la pagina, es bastante sencillo todo, vimos como poner colores de mil maneras y que al final solo ocupare dos xd, hacer cajas que no son cajas, un par de animaciones medio extrañas, como hacer diferentes tipos de textos que la verdad no le encuentro mucha utilidad, quiero ya empezar a programar de nuevo, java script te estoy esperando, pero php tu no, tu tas feito...
